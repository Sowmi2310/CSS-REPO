HTML Terms

HTML
HTML is used to define the structure of the web page.It expand as Hyper Text Markup Language
HTML element
Inline element
Block level element
HTML tag
HTML tag is a piece of markup Language used to indicate the beginning and end of an HTML element in an HTML document
HTML attribute
HTML attributes provide additional information about elements
HTML comment
The comment tag is used to insert comments in the source code,which can help our when we wish to edit the source code at a later date
HTML object element
The object tag defines a container for an external resource
HTML special character
Special characters are typically those that can't be easily typed into a keyboard or may cause display issues if typed or pasted into a web page
